Frontend Layer

React SPA with React Router v6. Handles UI, form state, JWT storage, and protected routing. Communicates only via Axios to the backend API.

Backend Layer

FastAPI with layered architecture: Router → Service → Model. Validates input with Pydantic. Generates JWT tokens. Hashes passwords with bcrypt.

Database Layer

MySQL stores user records. SQLAlchemy ORM maps Python classes to tables. Tables are auto-created at startup. Passwords are stored as bcrypt hashes — never plain text.
